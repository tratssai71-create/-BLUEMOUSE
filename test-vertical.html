縦画像テスト

縦長写真（ポートレート）を詳細ページ内でどのように使えるか、 4パターンのレイアウトで確認するテストページです。

Layout A

縦画像を左に・テキスト右

自然な縦比率を活かす

縦画像を左側に固定幅で配置し、テキストを右に展開するパターンです。人物写真など顔が入る縦構図に向いています。余白が生まれるので落ち着いた印象になります。

Layout B

カード型・画像上部

縦比率そのままカード

aspect-ratio: 3/4 で縦長カードにテキストを続けるパターン。縦画像がそのまま活きます。

2カラム並べも可能

2枚並べると写真集のような自然な見た目になります。複数の縦写真がある場合に効果的です。

Layout C

テキスト左・画像を右にトリミング

横帯レイアウトで縦画像を右配置

テキストを左側に広く取り、縦画像を右に配置してクロップするパターンです。横向きに並べるので、縦画像でもスッキリした横帯になります。見出しや説明量が多い場合に向いています。

Layout D

縦画像を大きく使う・ダーク背景

Feature

縦画像を半分・テキストを半分

左半分に縦画像を大きく表示し、右半分にダーク背景でテキストを並べるパターンです。インパクトがあり、メインビジュアルとしての活用にも向いています。

このページはレイアウトテスト用です。
